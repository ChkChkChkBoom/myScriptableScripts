<h1>myScriptableScripts</h1>
This repo has various scripts for <b>Scriptable</b>.<p>
Scriptable is an app that has similar functionality to Apple Shortcuts, but uses JavaScript.<p>
To use these scripts:
<ol>
  <li>Install <a href="https://apps.apple.com/us/app/scriptable/id1405459188">Scriptable</a> from the app store</li>
  <li>Create a new script, call it RepoManager</li>
  <li>Copy and paste RepoManager.js (from this repo) into the script you made</li>
  <li>Run it and select "install all" or select a script you want and then "install"</li>
  <li>Check bookmarks.js for more info on setup</li>
</ol>
